## 5/3/1 Boring But Big (BBB)

### Name
5/3/1 Boring But Big (BBB)

### Author/Source
**Jim Wendler**
- [Official Article](https://www.jimwendler.com/blogs/jimwendler-com/101077382-boring-but-big)
- [3-Month Challenge](https://www.jimwendler.com/blogs/jimwendler-com/boring-but-big-3-month-challenge)

### Description
Boring But Big is the most popular assistance template for Jim Wendler's 5/3/1 training program. It combines the proven 5/3/1 progression scheme with high-volume supplemental work to build both strength and size. The program earned its name from the monotonous yet effective 5x10 supplemental sets performed after the main work.

The philosophy is simple: get strong with heavy weights using the 5/3/1 core work, then build muscle mass with the volume-focused BBB work. The program is designed to be sustainable long-term, with built-in deload weeks preventing overtraining. It's ideal for intermediate lifters who want to add muscle mass while continuing to build maximal strength.

BBB is particularly effective because it forces lifters to perform a high volume of work with the main barbell movements (squat, bench press, deadlift, overhead press), which are the most effective exercises for building overall strength and size. The simplicity of the program makes it easy to follow consistently, which is the key to long-term progress.

### Structure
- **Program Type**: Linear periodization with wave-loading
- **Duration**: 4-week cycles (mesocycles) that can be repeated indefinitely
- **Training Days**: 3-4 days per week
- **Session Structure**: One main lift per session with 5/3/1 progression, followed by 5x10 supplemental work

### Session Breakdown

#### Week Structure
Each 4-week cycle follows this pattern:
- **Week 1 - 5s Week**: 65%, 75%, 85% (5+ reps on top set)
- **Week 2 - 3s Week**: 70%, 80%, 90% (3+ reps on top set)
- **Week 3 - 5/3/1 Week**: 75%, 85%, 95% (1+ rep on top set)
- **Week 4 - Deload**: 40%, 50%, 60% (no AMRAP sets)

#### Day 1: Squat Day
**Main Work (5/3/1):**
- Warm-up sets (40% x 5, 50% x 5, 60% x 3)
- Working sets based on week (see week structure above)
- Sets: 3 working sets
- Rest: 3-5 minutes between working sets

**BBB Supplemental Work:**
- Squat: 5 sets x 10 reps @ 50-60% Training Max
- Rest: 2-3 minutes between sets

**Assistance Work (50-100 total reps each):**
- Push: Dips, push-ups, or overhead press variations
- Pull: Chin-ups, pull-ups, or rows
- Single-leg/Core: Lunges, step-ups, or ab work

#### Day 2: Bench Press Day
**Main Work (5/3/1):**
- Warm-up sets (40% x 5, 50% x 5, 60% x 3)
- Working sets based on week
- Sets: 3 working sets
- Rest: 3-5 minutes between working sets

**BBB Supplemental Work:**
- Bench Press: 5 sets x 10 reps @ 50-60% Training Max
- Rest: 2-3 minutes between sets
- Can be supersetted with lat work (rows or pull-ups) to save time

**Assistance Work (50-100 total reps each):**
- Push: Dumbbell press variations, dips
- Pull: Rows (barbell, dumbbell, cable), lat pulldowns
- Single-leg/Core: Split squats, planks, ab wheel

#### Day 3: Deadlift Day
**Main Work (5/3/1):**
- Warm-up sets (40% x 5, 50% x 5, 60% x 3)
- Working sets based on week
- Sets: 3 working sets
- Rest: 3-5 minutes between working sets

**BBB Supplemental Work:**
- Deadlift: 5 sets x 10 reps @ 50-60% Training Max
- Rest: 2-3 minutes between sets
- **Important**: Start very light (30-40% TM) for first cycle

**Assistance Work (50-100 total reps each):**
- Push: Overhead press, incline press
- Pull: Chin-ups, face pulls
- Single-leg/Core: Leg curls, hip thrusts, ab work

#### Day 4: Overhead Press Day
**Main Work (5/3/1):**
- Warm-up sets (40% x 5, 50% x 5, 60% x 3)
- Working sets based on week
- Sets: 3 working sets
- Rest: 3-5 minutes between working sets

**BBB Supplemental Work:**
- Overhead Press: 5 sets x 10 reps @ 50-60% Training Max
- Rest: 2-3 minutes between sets
- Can be supersetted with lat work

**Assistance Work (50-100 total reps each):**
- Push: Dips, close-grip bench press
- Pull: Rows, pull-ups, shrugs
- Single-leg/Core: Bulgarian split squats, hanging leg raises

### Progression
- **Training Max Increases**: After each 4-week cycle:
  - Upper body lifts (Bench, OHP): +5 lbs
  - Lower body lifts (Squat, Deadlift): +10 lbs
- **BBB Weight Progression**:
  - Cycle 1-2: 50% of Training Max
  - Cycle 3-4: 55% of Training Max
  - Cycle 5-6: 60% of Training Max
  - Advanced: 65-70% of Training Max (only after 6+ months)
- **Deload Strategy**: Every 4th week, use deload weights (40/50/60%) with no AMRAP sets
- **When to Reset**: If you cannot achieve 3-5 reps on the 1+ set (Week 3), reduce Training Max by 10% and rebuild

### Program Duration
- **Single Cycle**: 4 weeks (3 weeks loading + 1 week deload)
- **Recommended Duration**: 6-12+ cycles (6-12+ months) for best results
- **Long-term Use**: Can be run indefinitely with proper deloads and progression

- [Boring But Big 3-Month Challenge](https://www.jimwendler.com/blogs/jimwendler-com/boring-but-big-3-month-challenge)

### Evidence Base
- **Author Credentials**: Jim Wendler is a former elite powerlifter who squatted 1,000 lbs in competition
- **Proven Track Record**: Used successfully by thousands of lifters worldwide for over 15 years
- **Scientific Principles**:
  - Progressive overload through percentage-based progression
  - Autoregulation via AMRAP sets
  - Adequate volume for hypertrophy (5x10 supplemental work)
  - Built-in fatigue management with deload weeks
  - Frequency: 2x per week per movement pattern (main + supplemental work)
- **Publications**: Featured in Wendler's books "5/3/1: The Simplest and Most Effective Training System for Building Raw Strength" and "Beyond 5/3/1"

### Database Implementation Notes
- **Exercises Present in Database**: Squat, Bench press, Deadlift, Overhead press (all core lifts are available)
- **Exercises Potentially Missing**:
  - Specific assistance exercises may need to be added based on athlete preferences
  - Dips, chin-ups, face pulls, Bulgarian split squats, ab wheel
- **Programming Considerations**:
  - Need to track Training Max separately from 1RM
  - AMRAP sets require rep tracking for progression decisions
  - Percentage-based calculations required
  - Deload weeks need special handling (no AMRAP)

## 5/3/1 for Beginners

### Name
5/3/1 for Beginners

### Author/Source
**Jim Wendler**
- [Official Article](https://www.jimwendler.com/blogs/jimwendler-com/5-3-1-for-beginners)
- [The Fitness Wiki Guide](https://thefitness.wiki/routines/5-3-1-for-beginners/)

### Description
5/3/1 for Beginners is a full-body training program designed for novice lifters who want to build strength using Jim Wendler's proven 5/3/1 methodology. Unlike standard 5/3/1 programs that train one main lift per session, this beginner variant trains two main lifts per workout across three weekly sessions, providing increased frequency for faster strength gains.

The program is built on the principle that beginners can recover faster and make gains more quickly than advanced lifters, so it includes more frequent exposure to each lift. By training each main movement 1.5 times per week on average, novices get the practice and stimulus needed to develop proper technique and build a strength base.

This program is particularly effective for new lifters because it balances structure with simplicity. The 5/3/1 progression provides a clear roadmap for long-term gains, while the full-body format prevents the common beginner mistake of neglecting certain movement patterns. The built-in First Set Last (FSL) supplemental work provides additional volume without the grinding fatigue of AMRAP sets on every working set.

### Structure
- **Program Type**: Linear periodization with full-body training
- **Duration**: 3-week cycles with no programmed deloads (deload when needed based on feel)
- **Training Days**: 3 days per week (Monday/Wednesday/Friday or similar spacing)
- **Session Structure**: Two main lifts per session (5/3/1 work + FSL supplemental work) plus assistance exercises

### Session Breakdown

#### Week Structure
Each 3-week cycle follows this pattern (percentages of Training Max):
- **Week 1 - 5s Week**: 65% x 5, 75% x 5, 85% x 5+
- **Week 2 - 3s Week**: 70% x 3, 80% x 3, 90% x 3+
- **Week 3 - 5/3/1 Week**: 75% x 5, 85% x 3, 95% x 1+

#### Day 1: Squat + Bench Press
**Squat - Main Work:**
- Warm-up: 40% x 5, 50% x 5, 60% x 5
- Working sets: Based on week (3 sets)
- Rest: 3-5 minutes between working sets

**Squat - FSL Supplemental:**
- 5 sets x 5 reps @ 65% (Week 1), 70% (Week 2), or 75% (Week 3)
- Rest: 2-3 minutes between sets

**Bench Press - Main Work:**
- Warm-up: 40% x 5, 50% x 5, 60% x 5
- Working sets: Based on week (3 sets)
- Rest: 3-5 minutes between working sets

**Bench Press - FSL Supplemental:**
- 5 sets x 5 reps @ 65% (Week 1), 70% (Week 2), or 75% (Week 3)
- Rest: 2-3 minutes between sets

**Assistance Work (50-100 total reps each category):**
- Push: Dips, push-ups, dumbbell bench press (50-100 reps total)
- Pull: Chin-ups, pull-ups, rows (50-100 reps total)
- Single-leg/Core: Lunges, split squats, leg raises (50-100 reps total)

#### Day 2: Deadlift + Overhead Press
**Deadlift - Main Work:**
- Warm-up: 40% x 5, 50% x 5, 60% x 5
- Working sets: Based on week (3 sets)
- Rest: 3-5 minutes between working sets

**Deadlift - FSL Supplemental:**
- 5 sets x 5 reps @ 65% (Week 1), 70% (Week 2), or 75% (Week 3)
- Rest: 2-3 minutes between sets

**Overhead Press - Main Work:**
- Warm-up: 40% x 5, 50% x 5, 60% x 5
- Working sets: Based on week (3 sets)
- Rest: 3-5 minutes between working sets

**Overhead Press - FSL Supplemental:**
- 5 sets x 5 reps @ 65% (Week 1), 70% (Week 2), or 75% (Week 3)
- Rest: 2-3 minutes between sets

**Assistance Work (50-100 total reps each category):**
- Push: Dips, close-grip bench, incline press (50-100 reps total)
- Pull: Barbell rows, dumbbell rows, lat pulldowns (50-100 reps total)
- Single-leg/Core: Bulgarian split squats, step-ups, ab wheel (50-100 reps total)

#### Day 3: Bench Press + Squat
**Bench Press - Main Work:**
- Warm-up: 40% x 5, 50% x 5, 60% x 5
- Working sets: Based on week (3 sets)
- Rest: 3-5 minutes between working sets

**Bench Press - FSL Supplemental:**
- 5 sets x 5 reps @ 65% (Week 1), 70% (Week 2), or 75% (Week 3)
- Rest: 2-3 minutes between sets

**Squat - Main Work:**
- Warm-up: 40% x 5, 50% x 5, 60% x 5
- Working sets: Based on week (3 sets)
- Rest: 3-5 minutes between working sets

**Squat - FSL Supplemental:**
- 5 sets x 5 reps @ 65% (Week 1), 70% (Week 2), or 75% (Week 3)
- Rest: 2-3 minutes between sets

**Assistance Work (50-100 total reps each category):**
- Push: Overhead press, dumbbell press variations (50-100 reps total)
- Pull: Chin-ups, face pulls, cable rows (50-100 reps total)
- Single-leg/Core: Walking lunges, hanging leg raises, planks (50-100 reps total)

### Progression
- **Training Max Increases**: After each 3-week cycle:
  - Upper body lifts (Bench, OHP): +5 lbs
  - Lower body lifts (Squat, Deadlift): +10 lbs
- **Faster Progression**: Increases every 3 weeks vs 4 weeks in standard 5/3/1
- **No Scheduled Deloads**: Take a deload week when you feel you need it (typically every 2-3 cycles)
- **When to Deload**:
  - Training Max feels too heavy
  - Recovery is becoming difficult
  - AMRAP sets yielding low rep counts
  - General fatigue or joint issues
- **Transition Point**: After 2-3 cycles (6-9 months), consider moving to standard 5/3/1 or 5/3/1 BBB for continued progress

### Program Duration
- **Single Cycle**: 3 weeks (no deload)
- **Recommended Duration**: 2-3 cycles before taking a deload (6-9 weeks)
- **Beginner Phase**: 6-12 months before transitioning to intermediate programming
- **Long-term Use**: Not recommended beyond 1 year; transition to standard 5/3/1 variants

- [The Philosophy for Beginners - Jim Wendler](https://www.jimwendler.com/blogs/jimwendler-com/101065094-5-3-1-for-a-beginner)

### Evidence Base
- **Author Credentials**: Jim Wendler, former elite powerlifter and strength coach
- **Scientific Principles**:
  - Higher frequency for skill acquisition (each lift trained ~1.5x per week)
  - Progressive overload through percentage-based progression
  - Autoregulation via AMRAP sets on top sets only
  - Sufficient volume for novice hypertrophy (FSL 5x5 work)
  - Full-body training optimizes recovery and anabolic environment for beginners
- **Success Rate**: Widely recommended by r/fitness community and strength coaches for novice lifters
- **Adaptability**: Proven effective across various populations (male, female, young, older beginners)

### Database Implementation Notes
- **Exercises Present in Database**: Squat, Bench press, Deadlift, Overhead press
- **Exercises Potentially Missing**:
  - Assistance exercise variety (dips, chin-ups, Bulgarian split squats, face pulls, ab wheel)
- **Programming Considerations**:
  - Need to track Training Max separately from 1RM
  - FSL percentages vary by week (65%/70%/75%)
  - AMRAP tracking only on final set of main work
  - No deload weeks in base program (taken as needed)
  - Assistance work is flexible (50-100 rep ranges per category)

## Tactical Barbell

### Name
Tactical Barbell

### Author/Source
**K. Black (KB Blackwell)**
- 20-year military and federal law enforcement veteran
- [Official Website](https://www.tacticalbarbell.com/)
- [Tactical Barbell I: Strength](https://www.amazon.com/Tactical-Barbell-Definitive-Strength-Operational/dp/1537666932)
- [Tactical Barbell II: Conditioning](https://www.amazon.com/Tactical-Barbell-2-Conditioning/dp/1537710443)

### Description
Tactical Barbell is a comprehensive strength and conditioning system designed specifically for operational athletes—military personnel, law enforcement, firefighters, and others whose jobs demand both strength and conditioning. Unlike pure strength or bodybuilding programs, Tactical Barbell recognizes that these athletes must maintain multiple physical qualities simultaneously: maximal strength, strength-endurance, and various conditioning capacities.

The program is built on the principle that operational athletes can't afford to sacrifice conditioning for strength or vice versa. The system uses sub-maximal training loads and structured periodization to build strength while maintaining the energy and recovery capacity needed for concurrent conditioning work. This makes it ideal for anyone who needs to be "strong enough" while also maintaining work capacity, endurance, and readiness.

The two-book system provides complete training guidance: Tactical Barbell I covers strength training with three main templates (Operator, Fighter, and Zulu) that can be customized with different exercise "clusters." Tactical Barbell II provides a three-pronged approach to conditioning, starting with Base Building to develop a foundation, then transitioning to continuation protocols (Black or Green) based on specific goals.

What sets Tactical Barbell apart is its emphasis on customization within a structured framework. Users select a strength template and exercise cluster based on their needs, time availability, and other training demands, then pair it with an appropriate conditioning protocol. This flexibility has made it popular not just with tactical athletes, but with anyone seeking concurrent strength and conditioning development.

### Structure
- **Program Type**: Block periodization with concurrent strength and conditioning
- **Components**:
  - Strength training (TB I) - Operator, Fighter, or Zulu templates
  - Conditioning training (TB II) - Base Building followed by Black or Green protocols
- **Training Frequency**: Varies by template (2-4 days/week for strength)
- **Session Structure**: Strength sessions separate from conditioning sessions

### Session Breakdown

#### Strength Templates Overview

**Operator Template** (Most Popular)
- Frequency: 3 days per week (minimum 48 hours between sessions)
- Cluster Size: 2-3 barbell exercises
- Sets/Reps: 3-5 sets of 1-5 reps
- Intensity: 70-95% of 1RM
- Periodization: Two 3-week waves (6 weeks total)

**Fighter Template**
- Frequency: 2 days per week
- Cluster Size: 4 exercises (all main lifts)
- Sets/Reps: 3-5 sets of 3 reps
- Intensity: ~90% of 1RM
- Best for: Maximum conditioning time availability

**Zulu Template**
- Frequency: 4 days per week (A/B/A/B split)
- Cluster Size: 4-6 exercises split into A and B workouts
- Sets/Reps: 3-5 sets of 1-5 reps
- Intensity: 70-95% of 1RM
- Best for: Maximum strength development

#### Operator Template - Detailed Breakdown

**Standard Operator Cluster Options:**
- Option 1: Squat, Bench Press, Deadlift
- Option 2: Squat, Overhead Press, Deadlift
- Option 3: Front Squat, Bench Press, Weighted Pull-up
- Option 4: Squat, Weighted Dip, Deadlift

**6-Week Block (Two 3-Week Waves):**

*Wave 1 - Weeks 1-3:*
- Week 1, Session 1: 3x5 @ 70%
- Week 1, Session 2: 3x5 @ 75%
- Week 1, Session 3: 3x5 @ 80%
- Week 2, Session 1: 3x3 @ 80%
- Week 2, Session 2: 3x3 @ 85%
- Week 2, Session 3: 3x3 @ 90%
- Week 3, Session 1: 5x2 @ 85%
- Week 3, Session 2: 5x2 @ 87.5%
- Week 3, Session 3: 5x2 @ 90%

*Wave 2 - Weeks 4-6:*
- Week 4, Session 1: 3x5 @ 75%
- Week 4, Session 2: 3x5 @ 80%
- Week 4, Session 3: 3x5 @ 85%
- Week 5, Session 1: 3x3 @ 85%
- Week 5, Session 2: 3x3 @ 90%
- Week 5, Session 3: 3x3 @ 92.5%
- Week 6, Session 1: 5x2 @ 87.5%
- Week 6, Session 2: 5x2 @ 90%
- Week 6, Session 3: 5x2 @ 92.5%

**Rest Periods:**
- 3-5 minutes between sets
- Minimum 48 hours between sessions

#### Fighter Template - Detailed Breakdown

**Standard Fighter Cluster:**
- Squat, Bench Press, Overhead Press, Deadlift (all 4 lifts each session)

**Weekly Structure:**
- Day 1 (Monday): All 4 lifts - 3-5 sets x 3 reps @ 90%
- Day 2 (Thursday): All 4 lifts - 3-5 sets x 3 reps @ 90%
- Rest: 3-5 days between sessions

**6-Week Block:**
- Weeks 1-6: Maintain same weight, focus on rep quality and speed
- After Week 6: Increase loads by 2.5-5% and repeat

#### Conditioning Component (TB II)

**Base Building (8 weeks):**
- Purpose: Build aerobic base and general conditioning
- Strength: Strength-Endurance circuits (2x/week)
- Endurance: Easy runs/rucks (3-4x/week)
- No max strength training during Base Building

**Continuation Protocols:**

*Black Protocol (Strength-focused):*
- Strength training: Operator/Fighter/Zulu (2-4x/week)
- HIC (High Intensity Conditioning): 1-2x/week
- Endurance: 1-2x/week (maintenance)

*Green Protocol (Endurance-focused):*
- Strength training: Fighter or Operator (2x/week)
- Endurance: 3-4x/week
- HIC: 1x/week

### Progression
- **Block-to-Block Progression**:
  - Complete 6-week block
  - Test new maxes (optional but recommended)
  - Increase training maxes by 2.5-5% for upper body, 5-10 lbs for lower body
  - Begin new 6-week block
- **Operator Progression**: Waves build from 70% to 92.5% over 6 weeks
- **Fighter Progression**: Maintain same weight for 6 weeks, then increase and repeat
- **Long-term Progression**: Continue until maxes stall, then take deload week and restart
- **Conditioning Progression**: Start with Base Building (8 weeks) then move to continuation protocols indefinitely

### Program Duration
- **Strength Blocks**: 6 weeks (two 3-week waves)
- **Base Building**: 8 weeks (run 1-2x per year)
- **Continuation**: Indefinite (run year-round with periodic Base Building phases)
- **Minimum Commitment**: 12 weeks (Base Building + one strength block)

  - Black vs Green protocol selection based on goals

## CrossFit Strength Bias

### Name
CrossFit Strength Bias (CFSB)

### Author/Source
**Jeff Martin and Darrell E. "Bingo" White**
- [CrossFit Journal Article (2009)](https://journal.crossfit.com/2009/02/crossfit-strength-bias.tpl)
- [Original PDF](https://library.crossfit.com/premium/pdf/CFJ_MartinWhite_StrengthBias.pdf)

### Description
CrossFit Strength Bias (CFSB) is a programming approach that adds systematic heavy strength training to the CrossFit methodology without sacrificing the sport's hallmark metabolic conditioning and gymnastics work. Developed by Jeff Martin and refined over five years with his athletes, CFSB addresses the common critique that standard CrossFit programming doesn't provide enough focused strength work for optimal strength development.

The philosophy stems from Greg Glassman's "What is Fitness" article, which described an ideal training session: warm up, perform 3-5 sets of 3-5 reps of a fundamental lift at a moderately comfortable pace, follow with a 10-minute circuit of gymnastics at high intensity, then finish with 2-10 minutes of metabolic conditioning. CFSB systematizes this approach with specific strength protocols paired with CrossFit WODs (Workouts of the Day).

The program is designed for CrossFit athletes who want to improve their strength numbers while maintaining the broad fitness base that defines CrossFit. By adding focused barbell work before the WOD, athletes can achieve meaningful strength gains that directly transfer to improved performance in CrossFit competitions and everyday functional movements. The key is balancing the strength work intensity and volume so that it enhances rather than interferes with the conditioning work.

CFSB has become a template used by many CrossFit affiliates and has influenced how competitive CrossFit athletes structure their training. It demonstrates that strength and conditioning can be developed concurrently when programmed intelligently.

### Structure
- **Program Type**: Concurrent strength and conditioning with strength emphasis
- **Training Days**: 4-5 days per week
- **Session Structure**: Heavy strength work (15-20 minutes) followed by CrossFit WOD (10-30 minutes)
- **Strength Focus**: At least 3 days per week of heavy barbell work
- **WOD Duration**: Primarily 10-20 minutes, rarely exceeding 30 minutes

### Session Breakdown

#### Weekly Template Structure

**Typical Week:**
- Monday: Heavy Squat + WOD
- Tuesday: Heavy Press + WOD
- Wednesday: Rest or light conditioning
- Thursday: Heavy Deadlift + WOD
- Friday: Heavy Olympic lift + WOD
- Saturday: WOD only (typically longer or heavier)
- Sunday: Rest

#### Strength Protocols

**5x5 Protocol:**
- 5 sets of 5 reps
- Goal: Work up to a heavy 5-rep max
- Suggested loads: 75-85% of 1RM
- Rest: 3-5 minutes between sets
- Use: Build strength endurance and volume

**3x5 and 5x3 "Choice Days":**
- Choose protocol based on daily feel and recovery
- 3x5: 3 sets of 5 reps @ 80-85% (when fresher)
- 5x3: 5 sets of 3 reps @ 85-90% (when more fatigued)
- Rest: 3-5 minutes between sets
- Goal: Post a PR weekly by adjusting between protocols

**5-5-5-5-5 Protocol:**
- 5 sets of 5 reps, increasing weight each set
- Final set should be a 5RM or near-5RM
- Rest: 3-5 minutes between sets

**3-3-3-3-3 Protocol:**
- 5 sets of 3 reps, increasing weight each set
- Final set should be a 3RM or near-3RM
- Rest: 3-5 minutes between sets
- Use: Heavier strength work

**1-1-1-1-1-1-1 Protocol:**
- 7 sets of 1 rep (or 5-7 heavy singles)
- Work up to daily max
- Not true 1RM testing (daily maximal effort)
- Rest: 3-5 minutes between sets

**12-Week Progressive Cycle:**
- Weeks 1-3: 3x8 (24 total work reps) @ 65-70%
- Weeks 4-6: 3x6 (18 total work reps) @ 70-75%
- Weeks 7-9: 4x4 (16 total work reps) @ 75-80%
- Weeks 10-12: 5x2 (10 total work reps) @ 85-90%
- Test maxes after week 12, then restart cycle

#### Sample Training Week

**Monday: Back Squat + WOD**
- Warm-up: 10 minutes (mobility, light movement)
- Strength: Back Squat 5-5-5-5-5 (work to heavy set of 5)
- Rest: 5-10 minutes
- WOD Example: "Fran" (21-15-9 Thrusters 95lbs/Pull-ups) for time

**Tuesday: Overhead Press + WOD**

## Mike Israetel's Hypertrophy Training Principles

### Name
Mike Israetel's Hypertrophy Training Principles (Volume Landmarks System)

### Author/Source
**Dr. Mike Israetel, Ph.D.**
- Co-founder and Chief Science Officer at Renaissance Periodization
- [Training Volume Landmarks Article](https://rpstrength.com/blogs/articles/training-volume-landmarks-muscle-growth)
- [Volume Landmarks Interactive Guide](https://volume-landmarks-rp-rals.vercel.app/)

### Description
Dr. Mike Israetel's Hypertrophy Training Principles represent a scientific framework for understanding and applying training volume for muscle growth. Rather than a specific program, these principles form the foundation for designing effective hypertrophy training. The system uses four volume landmarks—MV, MEV, MAV, and MRV—which define the relationship between training volume and muscle growth for each individual and each muscle group.

This framework recognizes that different people and different muscle groups have varying volume needs. For example, calves might need 16-20 sets per week to grow optimally, while biceps might only need 12-16 sets. The landmarks also acknowledge that volume needs change over time as muscles adapt, requiring progressive increases in volume within a mesocycle to maintain growth stimulus.

What makes this system powerful is its precision and individualization. Instead of generic "3 sets of 10" prescriptions, lifters learn to identify their personal MEV (minimum to grow), train within their MAV range (optimal growth), and avoid exceeding their MRV (beyond recovery capacity). This prevents both under-training (insufficient stimulus) and over-training (exceeding recovery).

The principles have revolutionized how evidence-based coaches and lifters think about programming. By understanding these landmarks, athletes can design programs that provide enough volume to grow without accumulating excessive fatigue. The system also explains why progressive overload must include volume progression, not just load progression, for continued hypertrophy.

### Structure
- **Framework Type**: Principles-based system, not a specific program
- **Application**: Can be applied to any hypertrophy program design
- **Core Concept**: Training within the MEV-MRV range, centering on MAV
- **Periodization**: Volume waves within mesocycles (MEV → MAV → approaching MRV → deload)

### Hypertrophy Training Principles

#### The Four Volume Landmarks

**1. MV (Maintenance Volume)**
- **Definition**: The minimum training volume needed to maintain current muscle mass
- **Typical Range**: 4-8 sets per muscle group per week (when training 2x/week)
- **Application**: Used during deloads, diet breaks, or when maintaining while focusing on other goals
- **Example**: If you stop training biceps but maintain 6 sets/week, your biceps won't shrink
- **Note**: Lower when training frequency is higher (distributed across more sessions)

**2. MEV (Minimum Effective Volume)**
- **Definition**: The minimum volume needed to make new muscle gains
- **Typical Range**: 8-12 sets per muscle group per week (varies significantly by muscle)
- **Application**: Starting point for new mesocycles after deload
- **Key Concept**: Anything below MEV maintains muscle; anything above MEV grows muscle
- **Individual Variation**: Beginners often have lower MEV; advanced lifters have higher MEV
- **Muscle Group Examples**:
  - Chest: 10-12 sets/week
  - Back: 12-14 sets/week
  - Quads: 12-16 sets/week
  - Hamstrings: 10-14 sets/week
  - Biceps: 8-12 sets/week
  - Triceps: 10-14 sets/week

**3. MAV (Maximum Adaptive Volume)**
- **Definition**: The volume range where optimal muscle growth occurs
- **Typical Range**: 12-20 sets per muscle group per week (highly individual)
- **Application**: The "sweet spot" for muscle growth—enough volume to maximize growth, not so much that fatigue interferes
- **Key Concept**: MAV is not a fixed number but a progressive range within a mesocycle
- **Progressive Nature**: Week 1 MAV might be 14 sets; Week 4 MAV might be 20 sets as you adapt
- **Muscle Group Examples**:
  - Chest: 14-18 sets/week
  - Back: 16-20 sets/week
  - Quads: 16-22 sets/week
  - Hamstrings: 12-18 sets/week
  - Shoulders: 14-18 sets/week
  - Biceps: 12-16 sets/week
  - Triceps: 14-18 sets/week
  - Calves: 16-24 sets/week

**4. MRV (Maximum Recoverable Volume)**
- **Definition**: The maximum volume you can handle while still recovering adequately
- **Typical Range**: 18-28 sets per muscle group per week (extremely variable)
- **Application**: The upper limit you approach in final weeks of mesocycle before deload
- **Key Concept**: Exceeding MRV leads to accumulated fatigue, joint stress, and stalled progress
- **Warning Signs**: Poor sleep, persistent soreness, performance decline, loss of motivation
- **Muscle Group Examples**:
  - Chest: 18-22 sets/week
  - Back: 20-25 sets/week
  - Quads: 20-28 sets/week
  - Hamstrings: 16-22 sets/week
  - Calves: 20-30 sets/week
  - Biceps: 16-20 sets/week
  - Triceps: 18-24 sets/week

#### Application Framework

**Mesocycle Structure (4-6 weeks):**
1. **Week 1**: Start at MEV (8-12 sets per muscle)
2. **Week 2**: Increase to lower MAV (12-16 sets per muscle)
3. **Week 3**: Increase to mid-MAV (16-20 sets per muscle)
4. **Week 4**: Approach MRV (20-25 sets per muscle)
5. **Week 5**: Deload (return to MV, 4-8 sets per muscle)

**Training Frequency:**
- 2x per week per muscle group: Standard recommendation
- 3x per week: Allows for more volume distribution, can push total weekly volume higher
- 1x per week: Only suitable for very high per-session volume (not recommended for most)

**Set Quality Requirements:**
- Sets must be taken to RPE 7-9 (1-3 reps from failure) to count as "working sets"
- Sets to failure (RPE 10) generate more fatigue and should be used sparingly
- Sets below RPE 6 are considered warm-up sets, not counted toward volume landmarks

**Exercise Selection:**
- Compound exercises: Count toward multiple muscle groups
  - Example: Bench press counts toward chest, front delts, triceps
- Isolation exercises: Count toward single muscle group
  - Example: Bicep curl only counts toward biceps
- Primary movers get full set count; secondary movers get 0.5-1 set count
  - Example: Bench press = 1 set for chest, 0.5 sets for triceps

**Rep Ranges for Hypertrophy:**
- Primary range: 6-20 reps per set
- Optimal growth range: 8-12 reps (balance of mechanical tension and metabolic stress)
- Strength-focused: 4-8 reps (lower volume per set, more sets needed)
- Metabolite-focused: 12-20+ reps (high metabolic stress, good for variation)
- Note: All rep ranges can build muscle when volume (sets x reps) is equated

**Rest Periods:**
- Compound lifts: 2-4 minutes
- Isolation exercises: 60-90 seconds
- Principle: Rest enough to perform next set with sufficient quality (rep target at target RPE)
- Shorter rest = more metabolic stress but may reduce mechanical tension
- Longer rest = better mechanical tension but longer workouts
