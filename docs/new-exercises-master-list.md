## Database Implementation Notes

### Exercises Requiring Verification
Some exercises may already exist under different names:
- **T-Bar Row** - DB has "T-Bar row" (check case sensitivity)
- **Seated Cable Row** - DB has "Seated pulley row" (likely same exercise)
- **Hack Squat** - DB has "Hack squat heavy/backoff" (verify if base exercise exists)
- **Dumbbell Step Up** - DB has this, but verify it's the same as programs require
- **Barbell Row** - DB has this, but may need specific variations (Pendlay, Bent-Over, etc.)

### Exercise Variations Needed
Several exercises need multiple variations tracked:
- **Pull-ups vs Chin-ups** (different grips)
- **Weighted vs Bodyweight** (pull-ups, dips, chin-ups)
- **Board Press** (2-board, 3-board, 4-board heights)
- **Box Squat** (various box heights)
- **Incline Press** (various angles: 15°, 30°, 45°, 60°)

### Exercise Aliases Recommended
To improve searchability, consider adding aliases:
- "Chin-ups" → alias: "Pull-ups (supinated grip)"
- "Romanian Deadlift" → alias: "RDL"
- "Lying Leg Curl" → alias: "Leg Curl", "Hamstring Curl (lying)"
- "Tricep Pushdowns" → alias: "Cable Pushdowns", "Tricep Extensions (cable)"
- "Face Pulls" → alias: "Rear Delt Cable Pull"
